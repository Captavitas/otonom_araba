# otonom_araba
Arduino tabanlı otonom araç projesi geliştirdim. Üç adet ultrasonik sensör ile engel algılama, LDR sensörüyle ışık değişimine duyarlı tünel geçişi ve motor kontrolü sağlayan sistem tasarladım. Araç, çevresindeki engelleri algılayarak yön değiştirebilmekte ve belirlenen senaryolara göre otonom kararlar alabilmektedir.
